\begin{abstract}
\noindent\textbf{Аннотация.} Кейс предназначен для формирования у студентов-радиотехников компетенций по применению аппарата рядов Фурье для спектрального анализа реальных сигналов, расчёта энергетических характеристик и инженерного обоснования параметров фильтров. Задания интегрируют математическую теорию с практикой проектирования радиотехнических устройств.
\end{abstract}

\section{Легенда (Инженерный контекст)}
В лаборатории проектирования радиопередающих устройств тестируется выходной каскад импульсного усилителя мощности. На осциллографе наблюдается периодическое напряжение $u(t)$, форма которого определяется режимом работы ключевых транзисторов и характеристиками выходного контура. Сигнал имеет период $T = 4$~мс и на интервале $[0; 2]$~мс описывается линейным спадом от максимального значения до нуля.

\medskip
\noindent\textbf{Инженеру-разработчику необходимо:}
\begin{enumerate}[label=\arabic*.]
    \item Представить сигнал в виде суммы гармонических колебаний (ряд Фурье).
    \item Оценить, какая часть мощности сосредоточена в постоянной составляющей и первых гармониках.
    \item Рассчитать полосу пропускания выходного фильтра нижних частот (ФНЧ), необходимую для передачи не менее 95\% полной мощности сигнала.
    \item Объяснить, как свойства симметрии сигнала влияют на спектр и требования к фильтрации.
\end{enumerate}

\section{Исходные данные}
\begin{itemize}[label=$\bullet$]
    \item Период сигнала: $T = 4$~мс $\Rightarrow l = \dfrac{T}{2} = 2$~мс.
    \item На отрезке $[0; l]$ напряжение задано функцией:
    \[
    u(t) = U_{\max} \left(1 - \frac{t}{l}\right), \quad U_{\max} = 12~\text{В}.
    \]
    \item Сопротивление нагрузки (для расчёта мощности): $R = 1$~Ом.
    \item Сигнал предполагается продолжить на $[-l; 0)$ \textbf{чётным образом} (типично для симметричных огибающих импульсов в радиотехнике).
\end{itemize}

\section{Задания для студентов}
\begin{enumerate}[label=\textbf{\arabic*.}, leftmargin=*]
    \item \textbf{Математическое моделирование.} Постройте чётное продолжение $u_{\text{чёт}}(t)$ на $[-l; l]$. Обоснуйте, почему в радиотехнической практике для огибающих импульсов чаще используют чётное, а не нечётное продолжение.
    
    \item \textbf{Расчёт коэффициентов Фурье.} Вычислите $a_0$ и $a_n$ по формулам для произвольного периода. Запишите тригонометрический ряд Фурье.
    
    \item \textbf{Анализ сходимости.} Оцените скорость убывания коэффициентов $a_n$. Свяжите её с гладкостью сигнала (теорема Дирихле). Будет ли ряд сходиться равномерно?
    
    \item \textbf{Энергетический анализ (Парсеваль).}
    \begin{enumerate}[label=\alph*), leftmargin=2em]
        \item Вычислите полную среднюю мощность сигнала $P_{\text{полн}}$.
        \item Найдите долю мощности, приходящуюся на постоянную составляющую ($n=0$), первую ($n=1$) и третью ($n=3$) гармоники.
        \item Определите, сколько гармоник необходимо передать для сохранения $\ge 95\%$ полной мощности.
    \end{enumerate}
    
    \item \textbf{Инженерное проектирование.} Рассчитайте частоту среза идеального ФНЧ, обеспечивающего передачу 95\% мощности. Укажите физическую размерность и обоснуйте выбор.
    
    \item \textbf{Визуализация и погрешность.} Постройте график частичной суммы $S_2(t)$ (постоянная + первые две ненулевые гармоники). Оцените максимальное отклонение от исходного сигнала. Обсудите, почему в данном случае эффект Гиббса отсутствует.
\end{enumerate}

\section{Методические рекомендации}
\begin{itemize}[label=$\triangleright$]
    \item Используйте формулы ряда Фурье для периода $2l$ из лекции (раздел~4).
    \item При интегрировании по частям внимательно следите за пределами и чётностью подынтегральных функций.
    \item Физическую среднюю мощность при $R=1$~Ом вычисляйте как 
    \[
    P_{\text{ср}} = \frac{1}{T} \int_0^T u^2(t) \, dt = \frac{1}{2l} \int_{-l}^l u^2(t) \, dt.
    \]
    \item Равенство Парсеваля в лекции дано в виде 
    \[
    \frac{1}{l} \int_{-l}^l f^2(x) \, dx = \frac{a_0^2}{2} + \sum_{n=1}^\infty (a_n^2 + b_n^2).
    \]
    Связь с мощностью: $P_{\text{ср}} = \dfrac{1}{2}\left(\dfrac{a_0^2}{2} + \sum a_n^2\right) = \dfrac{a_0^2}{4} + \dfrac{1}{2}\sum a_n^2$.
    \item Частота $n$-й гармоники: $f_n = \dfrac{n}{T} = n \cdot 250$~Гц.
\end{itemize}

\section{Подробное решение (ключевые этапы)}

\subsection{Чётное продолжение}
\[
u_{\text{чёт}}(t) = U_{\max}\left(1 - \frac{|t|}{l}\right) = 12\left(1 - \frac{|t|}{2}\right), \quad t \in [-2; 2]~\text{мс}.
\]
В радиотехнике чётное продолжение соответствует сигналам без фазового сдвига относительно начала отсчёта, что упрощает синтез квадратурных модуляторов и снижает уровень нечётных гармонических искажений в симметричных схемах.

\subsection{Коэффициенты Фурье}
Для чётной функции $b_n = 0$. Вычисляем $a_n$:
\[
a_0 = \frac{2}{l} \int_0^l u(t) \, dt = \int_0^2 12\left(1 - \frac{t}{2}\right) dt = \Bigl[12t - 3t^2\Bigr]_0^2 = 12.
\]
\[
a_n = \frac{2}{l} \int_0^l u(t) \cos\frac{\pi n t}{l} \, dt = \int_0^2 12\left(1 - \frac{t}{2}\right) \cos\frac{\pi n t}{2} \, dt.
\]
Интегрирование по частям ($u=12-6t$, $dv=\cos\frac{\pi n t}{2}dt$):
\[
a_n = \frac{24}{\pi^2 n^2}\bigl(1 - (-1)^n\bigr) = 
\begin{cases}
0, & n = 2k, \\[4pt]
\dfrac{48}{\pi^2 (2k+1)^2}, & n = 2k+1.
\end{cases}
\]
Ряд Фурье:
\[
u(t) = 6 + \frac{48}{\pi^2} \sum_{k=0}^{\infty} \frac{1}{(2k+1)^2} \cos\left(\frac{\pi (2k+1) t}{2}\right).
\]

\subsection{Анализ сходимости}
Коэффициенты убывают как $1/n^2$. Сигнал непрерывен на $[-l;l]$ и имеет кусочно-непрерывную производную (излом в $t=0, \pm l$). По теореме Дирихле ряд сходится к $u(t)$ во всех точках, причём \textbf{равномерно}. Скорость убывания $1/n^2$ характерна для сигналов без разрывов, что гарантирует быструю сходимость и малое внеполосное излучение.

\subsection{Энергетический анализ (Парсеваль)}
Полная средняя мощность ($R=1$~Ом):
\[
P_{\text{полн}} = \frac{1}{2l} \int_{-l}^l u^2(t) \, dt = \int_0^2 \bigl(12-6t\bigr)^2 dt = 36 \int_0^2 (2-t)^2 dt = 36 \cdot \frac{8}{3} = 48~\text{Вт}.
\]
Через коэффициенты (проверка Парсеваля):
\[
P_{\text{полн}} = \frac{a_0^2}{4} + \frac{1}{2}\sum_{k=0}^\infty a_{2k+1}^2 = 36 + \frac{1}{2} \cdot \frac{2304}{\pi^4} \sum_{k=0}^\infty \frac{1}{(2k+1)^4}.
\]
Используя $\displaystyle\sum_{k=0}^\infty \frac{1}{(2k+1)^4} = \frac{\pi^4}{96}$, получаем $P_{\text{полн}} = 36 + 24 \cdot \frac{1}{2} \cdot 8 = 48$~Вт. (Совпадение подтверждает корректность расчётов).

\noindent\textbf{Вывод:} Для передачи $\ge 95\%$ мощности достаточно передать постоянную составляющую и \textbf{первую гармонику}.

\subsection{Частота среза ФНЧ}
Первая ненулевая переменная гармоника имеет частоту:
\[
f_1 = \frac{1}{T} = \frac{1}{4 \cdot 10^{-3}} = 250~\text{Гц}.
\]
Для передачи 95\% мощности достаточно полосы $0 \div 250$~Гц. На практике берут запас $10$--$20\%$: $f_{\text{ср}} \approx 300$~Гц. Более широкий фильтр не нужен, так как высшие гармоники несут менее $0.1\%$ мощности и лишь ухудшают КНИ (коэффициент нелинейных искажений).

\subsection{График и погрешность}
Частичная сумма: $S_1(t) = 6 + \dfrac{48}{\pi^2} \cos\left(\dfrac{\pi t}{2}\right)$.

Максимальная погрешность наблюдается в точках излома ($t=\pm 2$): 
$|u(\pm 2) - S_1(\pm 2)| = |0 - (6 - 4.865)| \approx 1.135$~В. Относительная ошибка $\sim 9.5\%$ от амплитуды, но быстро снижается при добавлении $n=3$.

\noindent\textbf{Эффект Гиббса отсутствует}, так как сигнал непрерывен (нет скачков). Колебания сходимости носят плавный характер и обусловлены конечностью числа гармоник.

\section{Инженерная интерпретация и выводы}
\begin{enumerate}[label=\arabic*.]
    \item \textbf{Симметрия и спектр:} Чётное продолжение даёт только косинусные члены. В радиоприёмных трактах это упрощает демодуляцию, так как все гармоники синфазны с опорным генератором.
    
    \item \textbf{Энергетическая эффективность:} 75\% мощности сосредоточено в постоянной составляющей (смещение рабочей точки транзистора). Только $\sim 25\%$ полезно для передачи информации. В реальных передатчиках это компенсируется разделительными конденсаторами (подавление $n=0$) и согласующими цепями.
    
    \item \textbf{Фильтрация:} Быстрое убывание $1/n^2$ позволяет использовать ФНЧ 2-го порядка с узкой полосой. Это снижает уровень гармонических помех, упрощает конструкцию фильтра и повышает КПД усилителя.
    
    \item \textbf{Связь с практикой:} Равенство Парсеваля в радиотехнике используется для расчёта бюджета мощности, оценки КНИ, проектирования спектрально-эффективных модуляций (например, OFDM, где мощность распределяется по множеству поднесущих).
\end{enumerate}

\section{Вопросы для защиты / дискуссии}
\begin{enumerate}[label=\textbf{\alph*)}, leftmargin=*, itemsep=0.5em]
    \item Как изменится спектр, если сигнал продолжить \textbf{нечётно}? Какие практические последствия это будет иметь для фазовых соотношений в приёмном тракте?
    
    \item Почему в современных цифровых коммуникациях (5G, Wi-Fi) используют прямоугольные импульсы, у которых коэффициенты Фурье убывают медленно ($1/n$)? Как инженеры компенсируют внеполосное излучение?
    
    \item Если в выходном каскаде появится насыщение (ограничение амплитуды), форма сигнала станет ближе к меандру. Как это повлияет на коэффициенты Фурье и требования к ФНЧ?
    
    \item Как равенство Парсеваля используется при проектировании антенных решёток и расчёте диаграммы направленности?
    
    \item При каких условиях тригонометрический ряд Фурье можно дифференцировать почленно? Как это связано с проектированием широкополосных усилителей?
\end{enumerate}
