::К1_В1:: Что называется ядром линейного оператора $\hat{A}$? {
    =Множество векторов $\vec{x}$, таких что $\hat{A}\vec{x} = \vec{0}$.
    ~Множество всех векторов $\vec{y}$, представимых в виде $\hat{A}\vec{x}$.
    ~Множество собственных векторов оператора.
    ~Размерность пространства $L$.
}

::К1_В2:: Какое условие является необходимым и достаточным для обратимости линейного оператора в конечномерном пространстве? {
    =$\det A \neq 0$
    ~$\det A = 0$
    ~$\text{rank } A < \dim L$
    ~$\text{Ker } \hat{A} \neq \{\vec{0}\}$
}

::К1_В3:: Чему равен дефект обратимого линейного оператора в пространстве размерности $n$? {
    =0
    =1
    =n
    ~Зависит от матрицы оператора.
}

::К1_В4:: Сформулируйте теорему о ранге и дефекте (Rank-Nullity Theorem). {
    =$\text{rank } \hat{A} + \text{def } \hat{A} = \dim L$
    ~$\text{rank } \hat{A} - \text{def } \hat{A} = \dim L$
    ~$\text{rank } \hat{A} \cdot \text{def } \hat{A} = \dim L$
    ~$\text{rank } \hat{A} = \text{def } \hat{A}$
}

::К1_В5:: Если $\text{Ker } \hat{A} = \{\vec{0}\}$, то оператор $\hat{A}$: {
    =Инъективен (взаимно однозначен).
    ~Сюръективен, но не обязательно инъективен.
    ~Необратим.
    ~Имеет нулевой определитель.
}

::К1_В6:: Что представляет собой образ линейного оператора $\text{Im } \hat{A}$? {
    =Линейную оболочку столбцов матрицы оператора $A$.
    =Линейную оболочку строк матрицы оператора $A$.
    =Множество решений системы $A\vec{x}=\vec{0}$.
    =Множество собственных значений матрицы $A$.
}

::К1_В7:: Линейный оператор $\hat{A}: L \to L$ называется обратимым, если: {
    =Существует $\hat{A}^{-1}$ такой, что $\hat{A}\hat{A}^{-1} = \hat{A}^{-1}\hat{A} = \hat{I}$.
    =Существует $\hat{A}^{-1}$ такой, что $\hat{A}\hat{A}^{-1} = \hat{0}$.
    =Для любого $\vec{x}$ выполняется $\hat{A}\vec{x} = \vec{x}$.
    =Матрица оператора симметрична.
}

::К1_В8:: Если оператор $\hat{A}$ необратим, то его ядро: {
    =Содержит ненулевые векторы ($\text{Ker } \hat{A} \neq \{\vec{0}\}$).
    =Состоит только из нулевого вектора.
    =Совпадает со всем пространством $L$.
    =Пусто.
}

::К1_В9:: Ранг линейного оператора равен: {
    =Размерности его образа ($\dim \text{Im } \hat{A}$).
    =Размерности его ядра ($\dim \text{Ker } \hat{A}$).
    =Количеству строк в матрице оператора.
    =Определителю матрицы оператора.
}

::К1_В10:: Для линейного оператора в пространстве $\mathbb{R}^n$ условие $\text{rank } \hat{A} = n$ эквивалентно тому, что: {
    =Оператор обратим.
    =Оператор имеет нетривиальное ядро.
    =Столбцы матрицы оператора линейно зависимы.
    =Образ оператора является собственным подпространством.
}
